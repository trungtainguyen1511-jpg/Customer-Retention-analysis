# Customer-Retention-analysis
Customer Retention Analysis using RFM Segmentation
This project analyzes customer retention behavior using the AdventureWorks 2022 database and the RFM (Recency, Frequency, Monetary) model.

# Objective
Identify loyal customers, customers at risk of churn, and high-value customers in order to support retention and revenue optimization strategies.

# Tools Used
Microsoft SQL Server
AdventureWorks 2022 database
Window Functions (NTILE, SUM OVER)
CTEs (Common Table Expressions)

# Key Analyses
RFM customer segmentation
Distribution of customer segments
Top 20 customers by revenue
Average Order Value (AOV) by segment
Identification of high-value At-Risk customers

# Main Insights
The Churn segment represented the largest customer group.
At-Risk customers had significantly higher average order values than Loyal customers.
Potential customers showed strong revenue potential despite lower purchase frequency.

# Business Recommendations
Prioritize win-back campaigns for high-value At-Risk customers.
Increase AOV for Loyal customers through bundling and upselling strategies.
Use automated communication for Churn customers to reduce retention costs.

Applying the RFM model in a business context

Translating analytical results into business recommendations
